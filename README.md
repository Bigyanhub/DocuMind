DocuMind is a full-stack learning platform built with the MERN stack.  
It allows users to upload study documents and interact with them through summaries, flashcards, quizzes, and a document-focused chat system.

## Key Features

- Secure authentication with JWT
- Upload and manage PDF study documents
- Embedded PDF viewer
- AI-powered document chat, summaries, and concept explanations
- Auto-generated flashcards and quizzes
- Progress tracking dashboard and analytics
- Responsive UI built with Tailwind CSS

## Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **AI:** Google Gemini API
- **Auth:** JWT
